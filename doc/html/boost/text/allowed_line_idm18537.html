Function template allowed_line

boost::text::allowed_line

Synopsis

// In header: <boost/text/line_break.hpp>


template<typename CPRange, typename CPIter> 
  unspecified allowed_line(CPRange & range, CPIter it);

Description

Returns the bounds of the smallest chunk of text that could be broken off into a line, searching from it in either direction, as a line_break_cp_view.

This function only participates in overload resolution if CPRange models the CPRange concept.
