    Returns the maximum size in bytes a rope_view can have. 

  13. template<typename Fn> void foreach_segment(Fn && f) const;

    Visits each segment s of the underlying unencoded_rope and calls f(s). Each segment is a value whose type models a CharIter iterator-range. Depending of the operation performed on each segment, this may be more efficient than iterating over [begin(), end()).

    Requires:

    Fn is an Invocable accepting a single argument whose begin and end model CharIter.

    Swaps *this with rhs.
