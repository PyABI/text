Function code_point_bytes

boost::text::v1::code_point_bytes

Synopsis

// In header: <boost/text/transcode_iterator.hpp>


constexpr int code_point_bytes(unsigned char first);

Description

Given the first (and possibly only) code unit of a UTF-8 code point, returns the number of bytes occupied by that code point (in the range [1, 4]). Returns a value < 0 if first is not a valid initial UTF-8 code unit.

This function is constexpr in C++14 and later.
