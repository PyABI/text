Function template code_point_units

boost::text::v1::code_point_units

Synopsis

// In header: <boost/text/transcode_iterator.hpp>


template<typename T> unspecified code_point_units(T first);

Description

Given the first (and possibly only) code unit of a UTF-16 code point, returns the number of code units occupied by that code point (in the range [1, 2]). Returns a value < 0 if first is not a valid initial UTF-16 code unit.

This function is constexpr in C++14 and later.
