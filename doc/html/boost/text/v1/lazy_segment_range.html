Description

Represents a range of non-overlapping subranges. Each subrange represents some semantically significant segment, the semantics of which are controlled by the NextFunc template parameter. For instance, if NextFunc is next_paragraph_break, the subranges produced by lazy_segment_range will be paragraphs. Each subrange is lazily produced; an output subrange is not produced until a lazy range iterator is dereferenced.
