Function template as_utf32

boost::text::v1::as_utf32

Synopsis

// In header: <boost/text/transcode_view.hpp>


template<typename Range> constexpr auto as_utf32(Range && r);

Description

Returns a utf32_view over the data in r. The view will transcode the data if necessary. R may be a null-terminated pointer, or a reference to such a pointer, of intergral type T, as long as sizeof(T) == 4.
